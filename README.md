# 🐙 tako

Minimal Git workflow CLI with LLM-generated commit messages.

## Install
```bash
npm install -g @tarunbtw/tako
```

## Setup

Grab a free Groq API key at [console.groq.com](https://console.groq.com).  
tako will ask for it on first run.

## Commands

| Command | What it does |
|---|---|
| `tako i` | initializes a new git repository |
| `tako p` | Push with auto commit message |
| `tako config` | view or update your stored API key |
